# Fashion MNIST Classification with CNN

## Descripción
Proyecto de Deep Learning enfocado en clasificación de imágenes utilizando redes neuronales convolucionales (CNN) sobre el dataset Fashion MNIST.

El modelo fue desarrollado con TensorFlow y Keras para clasificar automáticamente prendas de vestir en 10 categorías.

---

## Tecnologías utilizadas

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

---

## Dataset

Fashion MNIST contiene imágenes de 28x28 píxeles en escala de grises pertenecientes a distintas categorías de ropa.

Clases:
- T-shirt/top
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle boot

---

## Arquitectura del modelo

La CNN implementada incluye:

- Capas convolucionales
- MaxPooling
- Dropout
- Flatten
- Capas densas
- Softmax para clasificación multiclase
